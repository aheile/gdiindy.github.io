Class 1:

1. Recap

Write a function addFive that takes a number as an argument, adds five to it, and returns the result. Write a function multiplyBySeven that returns its argument times seven, and finally, write a function addFiveThenMultiplyBySeven that first adds five to its argument, then multiplies the whole thing by seven, and returns the result. Challenge: use the first two functions in the last one instead of math operators.

2. Reflexion

Write a function concatTwo that takes two arguments and returns a string of those two aruments concatenated as strings. Note that it must support arguments that are either strings or numbers, and always return a string.

3. Challenge

Correct the 6 Javascript errors/mistakes in the code below to get a working greetings function that takes a first and last name as arguments (like 'Marie' and 'Curie') and returns a greeting string (like 'Welcome, Marie Curie!!!').
